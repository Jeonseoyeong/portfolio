(Q&A)BEHIND
MY DESIGN

( 01 ) 본인을 하나의 키워드로 소개하면 무엇인가요?
저는 솔직하고 분명한 소통을 중시하는 'Straight Communicator'라고 정의하고 싶습니다.
( 02 ) 나만의 강점과 약점은 무엇인가요?
저는 제가 좋아하고 싫어하는 것을 비교적 명확히 알고 있습니다.
그로 인해 선택에 망설임이 적은 편이지만, 익숙한 방향으로 판단이 좁아질 수 있다는 점을 인지하고 다양한 가능성을 함께 검토하려고 합니다.
( 03 ) 지금까지 진행한 프로젝트 중 가장 기억에 남는 프로젝트는 무엇인가요?
가장 기억에 남는 프로젝트는 'Nuvie' 팬덤 앱 프로젝트입니다.
팬덤 문화에 대한 이해가 부족한 상태에서 시작하며 여러 시행착오를 겪었고, 그 경험을 통해 설계의 출발점은 사용자라는 점을 배웠습니다.
( 04 ) 창의적인 아이디어를 얻는 나만의 방법이 있나요?
저는 특별한 장소를 찾기보다, 일상을 관찰하며 아이디어를 얻는 편입니다.
길거리의 표지판과 간판 타이포, 사람들의 옷에 담긴 패턴까지 저에게는 모두 영감의 재료가 됩니다.
( 05 ) 다른 사람과 협업할 때 중요하게 생각하는 점은 무엇인가요?
제가 생각하기에 협업에서 중요한 점은 상대와 제가 다를 수 있다는 전제를 가지고 대화하는 태도입니다.
